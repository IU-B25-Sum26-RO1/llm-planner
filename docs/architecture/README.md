# Документация архитектуры

На данном этапе проект имеет только часть для переработки голосового запроса в команду формата JSON, следственно файловая схема проекта выглядит так: 

```
Project 
├── src 
│   ├── audio_processor
│   ├── camera_driver
│   ├── decomposer
│   ├── robot_interfaces
│   ├── sam3_bridge
│   ├── sam3_preprocessor
│   ├── ur10e_control_system
│   └── ur10e_scene
├── docs
├── prompts
├── models
└── schemas
```
